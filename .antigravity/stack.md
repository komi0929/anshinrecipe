# Technology Stack
* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS (Mobile First)
* **Icons:** Lucide React
* **Backend/Auth:** Supabase
* **State Management:** React Hooks (Avoid Redux/Zustand unless necessary)
* **Data Fetching:** Server Components / Server Actions (No useEffect for data fetching if possible)
